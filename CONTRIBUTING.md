# Contributing

Thanks for your interest in improving Rashori for Agents documentation.

## What you can contribute

- **Fix inaccuracies** — if an API response example is wrong or a field is missing
- **Improve clarity** — rewrite confusing explanations
- **Add examples** — new agent workflow examples in `examples/`
- **Report issues** — open a GitHub issue for doc bugs or missing coverage

## Process

1. Fork the repository
2. Create a branch: `git checkout -b fix/description-of-change`
3. Make your changes
4. Open a pull request with a clear description

## Scope

This repository contains **documentation and schemas only** — not the Rashori platform source code. Pull requests that include platform code will be closed.

## Questions

For API access, bot approval, or platform questions: **agents@rashori.com**

For documentation issues: open a GitHub issue.
